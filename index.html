le HTML c'est quoi ?

HTML (HyperText Markup Language) n'est pas un langage de programmation : c'est un langage de balisage qui sert à indiquer au navigateur comment structurer les pages web visitées. Il peut être aussi compliqué ou aussi simple que le développeur web souhaite qu'il soit. Le HTML se compose d'une série d'éléments avec lesquels vous pouvez encadrer, envelopper ou baliser différentes parties du contenu pour les faire apparaître ou agir d'une certaine manière. Des balises encadrantes peuvent transformer une petite partie de contenu en un lien vers une autre page sur le Web, mettre des mots en italique, etc.

Les balises

Le HTML utilise des balises ouvrante et fermante <> pour que le navigateur puisse identifier l'élement.


Balise

    • Les balises:
    • <h1>
      la balise h1 permet de faire un titre
    • <p>
      la balise p permet de faire un paragraphe
    • <img>
      la balise img permet d'inserer une image
    • Les sélecteurs en css:
    • <class="...">
      La classe CSS est un attribut qui permet de sélectionner plusieurs éléments HTML et de définir leur style simultanément.
    • <id="...">
      définit un identifiant qui doit être unique pour l'ensemble du document

Exemple de balise

    • Ecrire un titre
    • Ecrire un paragraphe
    • Insérer une image en html
    • <h1> Je suis un titre </h1>
    • <p> Je suis un paragraphe </p>
    • <img src="URL de l'image" alt=" description textuelle que l’on attribue à une image.">

Les balises CSS

    • modifier une couleur
    • modifier la taille d'un texte
    • Ajouter une bordure
    • p {color: red}
    • p {font-size:1.1em}
    • p {border: solide 1px}
